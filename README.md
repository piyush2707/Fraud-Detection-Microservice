# Fraud-Detection-Microservice
 Project Summary
This project showcases an end-to-end MLOps pipeline built to deploy a real-time credit card fraud detection model.
The primary goal was to create a production-ready system by integrating a trained XGBoost Classifier into a scalable, containerized web API.
